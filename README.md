# Cosmo Runner Web3 Telegram Bot

Мини-приложение Telegram с неоновым раннером в стиле оффлайн-игры Chrome. Проект
включает бота, который выдает кнопку для запуска WebApp, Express-сервер для
статической раздачи и простую таблицу лидеров, а также фронтенд с поддержкой
TON Connect для связывания вашего кошелька.

## Возможности

- 🚀 **Web3 WebApp** — кнопка запуска в Telegram, работающая на любом устройстве.
- 🎮 **Неоновый раннер** — прыжки, двойной прыжок, динамические препятствия
  (кристаллы, деревья, порталы) и неоновая визуализация.
- 🔥 **Комбо-баннеры** — уведомления при серии успешных прыжков без столкновений.
- 🏆 **Онлайн-таблица лидеров** — результаты сохраняются на сервере и доступны
  всем пользователям.
- 💎 **TON Connect** — подключите кошелек, чтобы закрепить ник в таблице.
- 🤖 **Telegraf бот** — команды `/start` и `/help`, поддержка `web_app_data`.

## Как запустить локально

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Создайте файл `.env` на основе `.env.example` и укажите токен бота Telegram,
   а также публичный URL, по которому будет доступно WebApp (например, ваш
   ngrok-адрес или домен).

3. Запустите сервер:

   ```bash
   npm run dev
   ```

   Приложение поднимет Express на `http://localhost:3000` и запустит Telegram
   бота.

4. В BotFather включите WebApp:

   - Выполните `/setdomain` и укажите домен/URL сервера.
   - Опционально назначьте меню или введите `/setmenubutton` → `web_app` → URL.

5. Откройте бота в Telegram, введите `/start` и нажмите кнопку **«🚀 Запустить
   космораннер»**.

## Конфигурация TON Connect

Для полноценной работы TON Connect необходимо задеплоить статические файлы и
обновить `public/tonconnect-manifest.json`, указав актуальные ссылки на иконку и
домен. Манифест должен быть доступен по HTTPS.

## Деплой

- Хостинг: Render, Railway, Fly.io, Vercel (через Node сервер) или любой VPS.
- Не забудьте выставить переменные окружения `BOT_TOKEN`, `WEB_APP_URL`, `PORT`.
- В Telegram BotFather установите `https://<ваш-домен>` как домен WebApp.
